# Rainfall Pattern Analysis & Prediction Using Machine Learning  

## Ziarat District, Balochistan

Developed a machine learning-based rainfall analysis and prediction model using historical data from 2020–2024. 
The project analyzes rainfall patterns for 2024 and forecasts future trends for 2026–2028. 
By combining data analysis and predictive modeling, the study aims to support climate insights, 
agricultural planning, water resource management, and disaster preparedness in Ziarat District, Balochistan.
